# README #

Este é um repositório com ROS packages para controle do quadrirrotor DJI Matrice 100.

Os seguintes packages estão disponíveis:

* first\_fly:
    + decola,
    + sobe até dois metros de altura,
    + vai para frente com a velocidade de 0.3 m/s por 3 segundos,
    + vai para trás com a velocidade de 0.3 m/s por 3 segundos e
    + pousa.
